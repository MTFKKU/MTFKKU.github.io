Efficient, Rapid, and Seamlessly automated.

MTF stands for Modulation Transfer Function, and KKU stands for Khon Kaen University.
Software to process MTF bars in order to automate this tedious task.

Windows

How to use?
Capture an X-ray image aligned with the MTF bar parameters, either longitudinally or transversely with the image detector. Avoid cropping the image manually; let the software handle it automatically.

Get an error?
If you get an error questioning the MTF bar's legitimacy, check if the DICOM file matches the tutorial video's MTF bar pattern. If not, the software won't process it. If errors persist despite a matching pattern, try making slight adjustments to the bar's direction.
