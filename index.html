How to use?

  • Capture an X-ray image aligned with the MTF bar.
  • Avoid cropping the image manually(software will handle it).
  • Open software then load image, click process.
  • Wait a second and done.

    •

    Get an error?

  • If you get an error questioning the MTF bar's legitimacy.
  • Check if the DICOM file matches the video's MTF bar pattern.
  • If not, the software won't process.
  • If errors persist despite a matching pattern.
  • Try making slight adjustments to the bar's direction.
